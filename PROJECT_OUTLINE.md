# SRTP项目工作大纲
# 多模态AI解读放射学报告对患者健康信念的影响研究

## 一、项目目标
探究多模态AI（结合影像+文本）解读放射学报告对患者健康信念的影响，基于健康信念模型(HBM)进行分析。

## 二、研究内容（调整后）

### 1. 基础理论与政策梳理
- [ ] 健康信念模型(HBM)理论深度梳理
- [ ] 社会技术系统理论(STS)
- [ ] 《卫生健康行业人工智能应用场景参考指引》政策分析
- [ ] 放射学AI和多模态AI医疗应用现状

### 2. 微观影响机制构建（HBM模型）
- [ ] 设计问卷（基于HBM维度：感知易感性、感知严重性、感知益处、行动线索、自我效能）
- [ ] 对比：多模态AI vs 纯文本LLM解读效果
- [ ] 分析AI解读对患者健康信念和就医行为的影响

### 3. 宏观双向作用分析（STS理论）
- [ ] 技术系统（准确性、可解释性、多模态优势）
- [ ] 社会子系统（医疗资源、医患关系、政策监管）

### 4. 关键挑战识别与优化路径
- [ ] 患者信任度问题
- [ ] 隐私安全顾虑
- [ ] 术语转化适配

### 5. 技术拓展：报告解读原型系统 ⭐新增
- [ ] 设计系统架构（前端展示 + 后端API）
- [ ] 实现纯文本LLM解读模块
- [ ] 实现多模态AI解读模块（影像+文本）
- [ ] 搭建对比展示界面
- [ ] 部署与测试

### 6. 技术拓展：访谈文本主题分析 ⭐新增
- [ ] 收集访谈文本数据
- [ ] 文本预处理（分词、去噪、标准化）
- [ ] LDA主题建模
- [ ] 情感分析与关键词提取
- [ ] 可视化与结果输出
