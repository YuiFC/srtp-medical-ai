## 近期行动项（2026年3月-4月）

### 1. 问卷预测试 ⏳ 待开展

**目标**：验证问卷信效度，收集修改意见

**执行步骤**：
1. 将问卷导入问卷星平台（使用 questionnaire_online_format.md）
2. 邀请10-15名目标人群试填
3. 收集反馈：题项是否清晰、是否有理解困难
4. 根据反馈修改问卷
5. 计算Cronbach's α系数，检验内部一致性

**预计时间**：1-2周

**负责人**：周昊

### 3. 正式调查实施 📅 计划4月启动

**目标**：收集200-300份有效问卷

**执行步骤**：
1. 发布问卷（问卷星/腾讯问卷）
2. 线上推广：微信群、朋友圈、问卷互填群
3. 线下发放：医院放射科、社区卫生服务中心
4. 每日监控问卷收集进度
5. 数据清洗和质量控制

**样本量目标**：200-300份

**预计时间**：4月-5月

---

### 4. 访谈实施 📅 计划5月启动

**目标**：获取质性数据，补充问卷调查

**执行步骤**：
1. 招募15-20名受访者
2. 准备访谈场地和设备（录音笔、录像设备）
3. 进行一对一访谈（20-30分钟）
4. 访谈录音转录
5. 主题分析法编码

**预计时间**：5月-6月

---

### 5. 数据分析 📅 计划6月启动

**目标**：完成统计分析

**执行步骤**：

3. 信度效度分析
4. 相关性分析
5. 回归分析
6. 结果可视化

**使用工具**：Python (pandas, scipy, statsmodels, pingouin)

**预计时间**：6月

## 风险与应对

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 问卷回收不足 | 可能影响统计分析 | 扩大推广渠道，增加激励 |
| 伦理审批延迟 | 影响调查开展 | 提前沟通，准备补充材料 |
| 访谈对象招募困难 | 质性数据不足 | 通过问卷受访者中招募 |
| 数据质量问题 | 影响分析结果 | 严格质量控制，剔除无效数据 |
