# 公开医疗数据集汇总

## 国际数据集

### 1. MIMIC系列（重症监护医疗数据）
- **MIMIC-III**: Medical Information Mart for Intensive Care
  - 来源：MIT + Beth Israel Deaconess Medical Center
  - 内容：去标识化的电子健康记录
  - 时间跨度：2008-2019
  - 链接：https://mimic.physionet.org/
  
- **MIMIC-IV**: 最新版本
  - 链接：https://www.nature.com/articles/s41597-022-01899-x

### 2. HealthBench
- 来源：OpenAI
- 内容：5000个真实健康对话
- 用途：评估AI在医疗场景中的表现
- 链接：https://www.ophthalmologyadvisor.com/news/openai-releases-healthbench-dataset-to-test-ai-in-health-care/

### 3. 医疗AI文献数据集
- 来源：PMC
- 内容：Health AI研究文献目录
- 链接：https://pmc.ncbi.nlm.nih.gov/articles/PMC10997486/

### 4. 其他公开数据集
- Kaggle医疗数据集
- AWS Open Data医疗数据集

## 数据集获取注意事项

1. **MIMIC数据集获取**：
   - 需要申请并通过考试
   - 免费获取，但需遵守使用协议
   
2. **数据隐私**：
   - 使用去标识化数据
   - 遵守相关法规

## 项目适用性分析

对于"AI解读医疗报告"项目：
- MIMIC更适合临床数据分析
- 需要找**患者可理解的医疗报告解读**相关数据集
- 可能需要自行收集问卷数据
