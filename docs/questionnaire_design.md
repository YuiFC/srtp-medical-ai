# 问卷设计文档
# 多模态AI解读放射学报告对患者健康信念的影响研究

---

## 一、问卷设计背景

### 1.1 研究目的
设计基于健康信念模型(HBM)的问卷，测量患者对AI解读放射学报告的接受度和健康信念。

### 1.2 理论基础
- **健康信念模型(HBM)**：通过聚焦人们的感知来解释和预测健康行为
- **社会技术系统理论(STS)**：关注AI技术与用户工作流程的整合

### 1.3 参考量表
- ATRAI-14放射科医生AI态度问卷
- 护理学生AI认知量表
- 中文版HBM量表
- 患者对AI医疗接受度问卷(JMIR)

---

## 二、问卷结构

### 2.1 量表维度与题目

#### 第一部分：人口统计学信息
1. 您的年龄：____岁
2. 您的性别：□男 □女 □其他
3. 您的学历：□初中及以下 □高中/中专 □大专 □本科 □硕士及以上
4. 您是否有过放射学检查(如CT、MRI、X光)经历？□是 □否
5. 您是否了解人工智能(AI)在医疗中的应用？□完全不了解 □不太了解 □一般 □比较了解 □非常了解

## 三、计分方法

- 采用Likert 5点量表：1=完全不同意，2=不同意，3=一般，4=同意，5=完全同意
- 感知障碍维度：得分越高表示障碍越大
- 其他维度：得分越高表示该信念越强

---

## 四、问卷长度与预计耗时

- 总题数：约35-40题
- 预计填答时间：10-15分钟

---

## 五、预测试计划

1. 邀请10-15名目标人群进行预测试
2. 收集反馈：题项是否清晰、是否有理解困难
3. 计算Cronbach's α系数，检验内部一致性
4. 根据反馈修改题项
