# 问卷预测试执行指南

## 预测试目的
- 验证问卷题项的清晰性和可理解性
- 检验问卷的信度（Cronbach's α > 0.70）
- 检验问卷的效度（因子分析）
- 评估问卷完成时间（目标：10-15分钟）
- 收集受访者反馈

## 预测试样本
- 样本量：10-15人
- 目标人群：有放射学检查经历的患者
- 纳入标准：年满18岁，中文阅读能力

## 预测试步骤

### 步骤1：问卷导入问卷星
1. 登录问卷星账号
2. 创建新问卷 → 手动创建
3. 按照 `questionnaire_online_format.md` 逐题导入
4. 设置题目逻辑（如第4题选"否"可跳转）
5. 设置问卷外观（标题、说明、结束语）

### 步骤2：发布预测试问卷
1. 设置问卷为"预测试"模式
2. 设置收集数量：15份
3. 分享问卷链接给目标人群
4. 邀请身边符合条件的人填写

### 步骤3：收集反馈
除填写问卷外，请预测试受访者额外回答：
1. 哪些题目难以理解？请说明
2. 哪些选项让您感到困惑？
3. 完成问卷大约用了多长时间？
4. 您对问卷整体长度是否满意？
5. 其他建议或意见

### 步骤4：数据分析
使用 `main_analysis.py` 分析预测试数据：
```bash
python main_analysis.py --data pretest_data.csv --output pretest_results
```

关注指标：
- Cronbach's α 系数（应 > 0.70）
- 各维度与总体的相关性
- 项删除后的α变化
- 受访者反馈汇总

### 步骤5：问卷修订
根据分析结果和受访者反馈：
- 删除区分度低的题项
- 修改表述不清的题项
- 调整选项设置
- 必要时调整维度结构

## 注意事项
1. 预测试样本应尽量覆盖不同年龄段、学历
2. 确保受访者有充足时间填答和反馈
3. 及时记录和分析反馈信息
4. 修订后的问卷需要再次小范围验证（如有必要）
