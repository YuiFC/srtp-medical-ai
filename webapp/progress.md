# SRTP 医疗AI报告解读原型 - 开发计划

## 项目概述
构建一个网页应用原型，展示AI解读放射学报告的功能。

## 功能需求
1. 上传医学影像（X光/CT）
2. 输入放射学报告文本
3. AI对比解读（多模态 vs 纯文本）
4. 健康信念问卷模块
5. 展示解读结果

## 技术栈
- Vue 3 + Vite
- Element Plus UI
- 后端: FastAPI (Python)
