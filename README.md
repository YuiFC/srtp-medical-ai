# SRTP项目：人工智能解读医疗报告的影响机制与挑战

## 项目目录结构
```
srtp_medical_ai/
├── WORK_LOG.md     # 工作记录
├── code/           # 代码目录
├── data/           # 数据目录
├── logs/           # 日志目录
├── archives/       # 存档目录（保留3份）
└── docs/           # 文档目录
```

## 研究框架
- 微观层面：健康信念模型(HBM) - 患者接受度与行为
- 宏观层面：社会技术系统理论(STS) - 技术与社会交互

## 技术路线
1. 前期准备：文献综述、框架构建
2. 数据收集：文本分析 + 问卷调查
3. 分析论证：统计分析与机制剖析
4. 成果输出：模型构建与优化路径
